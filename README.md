
# Flutter Music Player

A simple android music player.



## Deployment

To deploy this project run

```bash
  flutter build apk --release
```


If you have connected android device, run

```bash
  flutter install
```



## Features

- Shuffle Mode
- Sorting
- Mini-Music Player
- Search Mode
