
# Flutter Music Player

It only works if you have an External Storage and it has mp3 related formats.



## Deployment

To deploy this project run

```bash
  flutter build apk --release
```


If you have connected android device, run

```bash
  flutter install
```



## Features

- Shuffle Mode
- Sorting
- Mini-Music Player
- Search Mode
